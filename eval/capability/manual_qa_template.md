# 人工问答抽检（能力套系补充）

自动化能力榜不覆盖业务质量。改配置或升级 vLLM 后，建议固定题单人工打 1～5 分。

| # | 类型 | 采样 | 检查点 |
|---|------|------|--------|
| 1 | 推理 | temp=1.0，思考开 | 有 think 块、结论合理 |
| 2 | 编码 | temp=0.6 | 代码可运行、无乱码 |
| 3 | 快答 | enable_thinking: false | 无 think 块、响应快 |
| 4 | 长文档 | 6 万字摘要 | HTTP 200、不 OOM |
| 5 | Agent 角色 | 固定长 system ×2 | 第二次明显更快（prefix） |

记录格式：

```text
快照 ID: prod-local-YYYYMMDD
题号 | 通过 | 备注
1    | Y/N  |
...
```
