# 第二轮调优结论（128K 细粒度矩阵）

数据来源：`results/r2/summary.csv`，14/14 变体 `OK`，全自动 P0+P1+P2 已完成。

## 推荐配置

| 用途 | 变体 | 说明 |
|------|------|------|
| **日常主力（推荐）** | **R2-A1** | BF16 KV + prefix + MTP×2；B2/B3 优于 T7b(A0)，保留官方 MTP num=2 |
| **长输出偏重** | **R2-A4** | MTP×4；B6 **82.95** tok/s（A0 为 73.08） |
| **回退** | **R2-A0 (T7b)** | FP8 KV；显存更省，吞吐略低 |
| **不建议默认** | A2/A3 | 关 MTP 后 B2 约 **-33%** |
| **不建议默认** | A5 | 无 prefix；长前缀场景损失大 |
| **谨慎** | B2/C3 | util=0.94 或 batched=16384；显存余量 &lt;3GB |

启动脚本：`configs/docker-run-main-128k-r2-final.sh`（对应 R2-A1）
